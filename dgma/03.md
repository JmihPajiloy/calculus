# 3. Предельная точка множества. Определение предела фнп (по Коши, по Гейне и через окрестности). Примеры

## Предел
### По Коши

$$
\forall \varepsilon > 0 \ \exists \delta(\varepsilon) > 0 : \forall P \in E \cap \mathring{U}_\delta(P_0) \Rightarrow |f(P) - A| < \varepsilon
$$

где $\mathring{U}_\delta(P_0)$ — проколотая окрестность точки $P_0 = (x_0, y_0)$.

### По Гейне

$$
\begin{cases}
x_n \to x_0, \ y_n \to y_0, \ (x_n, y_n) \ne (x_0, y_0) \\
\Rightarrow f(x_n, y_n) \to A
\end{cases}
$$

### Через окрестности (в координатах)

$$
\begin{cases}
0 < |x - x_0| < \delta \\
0 < |y - y_0| < \delta
\end{cases}
\Rightarrow |f(x,y) - A| < \varepsilon
$$
